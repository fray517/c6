# Калькулятор — деплой на Vercel

Простой статический проект с файлом `calculator.html`, подготовленный
для развёртывания на Vercel.

## Требования

- Установлен Node.js
- Установлен pnpm
- Установлен Vercel CLI (через pnpm dlx)

## Локальный превью и деплой

1. В PowerShell перейдите в папку проекта:

   ```powershell
   Set-Location C:\Work\c6
   ```

2. Запустите интерактивную настройку проекта в Vercel (один раз):

   ```powershell
   pnpm dlx vercel
   ```

   - При запросе пути к проекту укажите текущую папку.
   - Когда спросят про команду сборки, выберите вариант без сборки
     (статический проект).

3. Для последующих деплоев используйте:

   ```powershell
   pnpm dlx vercel --prod
   ```

Проект настроен так, что корневой путь `/` отдаёт страницу
`calculator.html` (см. конфиг `vercel.json`).
